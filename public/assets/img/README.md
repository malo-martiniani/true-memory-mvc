# Images pour le jeu de mémoire

Ce dossier contient les images utilisées pour les cartes du jeu de mémoire.

## Images nécessaires

Le jeu prend en charge de 3 à 12 paires de cartes. Vous devez fournir 12 images nommées comme suit :

- `card-1.jpg`
- `card-2.jpg`
- `card-3.jpg`
- `card-4.jpg`
- `card-5.jpg`
- `card-6.jpg`
- `card-7.jpg`
- `card-8.jpg`
- `card-9.jpg`
- `card-10.jpg`
- `card-11.jpg`
- `card-12.jpg`

## Format recommandé

- Format : JPG, PNG ou GIF
- Dimensions : Carrées (par exemple 200x200px, 300x300px)
- Taille fichier : Optimisées pour le web (< 100KB chacune)

## Notes

- Les images doivent être placées directement dans ce dossier (`/public/assets/img/`)
- Le jeu affichera ces images lorsque les cartes seront retournées
- Si une image n'est pas trouvée, un numéro sera affiché à la place
